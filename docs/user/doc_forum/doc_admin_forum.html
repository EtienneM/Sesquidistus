Catégories

Administration du forum:

  • Ajouter une catégorie

Ajouter une catégorie:

Pour ajouter une catégorie au forum, cela se passe en deux étapes:

Tout d'abord, rendez-vous sur la page d'administration du forum en cliquant sur l'icône adéquate dans le panel d'administration du site. Ensuite sélectionnez le traitement "Ajouter une catégorie" dans la liste déroulante des traitements comme ceci :

Maintenant remplissez les champs avec les valeurs que vous désirez. La valeur de la position affichée, correspond la dernière place sur le forum. Libre à vous de positionner votre nouvelle catégorie où bon vous semble. Puis cliquez sur le bouton valider, un message de confirmation vous informera du bon déroulement de l'opération.

Supprimer un sujet:

Pour supprimer un sujet du forum, cela se passe en deux étapes:

Tout d'abord, rendez-vous sur la page d'administration du forum en cliquant sur l'icône adéquate dans le panel d'administration du site. Ensuite sélectionnez le traitement "Supprimer un sujet" dans la liste déroulante des traitements comme ceci :

Maintenant selectionnez la sous-catégorie contenant le sujet à supprimer dans la liste déroulante puis selectionnez le sujet par la liste de sujet proposés. Puis cliquez sur le bouton valider, une demande de confirmation apparaitra à l'écran. Enfin une fois la suppression confirmée, un message de confirmation vous informera du bon déroulement de l'opération.

Exemple de suppression de sujet: (Suppression du sujet "Mon premier..." dans la sous catégorie "Ma sous-catégorie 1")
