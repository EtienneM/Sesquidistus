  • Ajouter un évènement
  • Modifier un évènement préalablement enregistré

Ajouter un évènement:

Pour l'ajout d'un évènement, veuillez suivre les étapes suivantes:

Type et nom:

Il s'agit dans un premier temps de définir un type et un nom à l'évènement que l'on souhaite enregistrer.




Lieu:

Il faut désormais définir un lieu dans lequelle se déroulera l'entrainement. Pour cela, il y a deux possibilités, soit sélectionner un lieu pré-enregistré:

Soit selectionner "liste des lieux" et saisir vous-même le lieu de l'évènement:




Durée:

L'étape suivante consiste à indiquer sur combien de jour(s) l'évènement se déroule. (exemple: Pour un tournoi se déroulant le week-end (samedi et dimanche) la durée sera de 2 (jours))

Vous pouvez enregistrer un nouveau lieu et son adresse (pour les lieux fréquemment utilisés) via cette page.



Horaires:

Maintenant, il s'agit de selectionner les horaires de début et de fin de l'évènement:

Si vous ne souhaitez pas définir d'horaire pour cet évènement, vous pouvez alors cocher la case se trouvant sur la droite du premier champ horaire:




Date:

Enfin, étape très importante, il est désormais nécessaire de selectionner la date de début de l'évènement parmis les dates selectionnables sur les trois calendrier:

Si vous sélectionner plusieurs dates, alors plusieurs évènements seront enregistrés avec les mêmes informations remplies précédemment (type, nom, lieu, durée, horaires).

Exemple: (Si vous souhaitez ajouter un tournoi qui aura lieu sur trois jours, il ne faut pas sélectionner 3 dates qui se suivent, il faut simplement donner la valeur 3 au champ "durée" et sélectionner la date de début du l'évènement dans un des calendrier).

Commentaire (facultatif):

Vous avez également la possibilité d'ajouter un commentaire à cet évènement qui sera visible aux membres.


Validation:

Il ne vous reste plus qu'à valider l'évènement en cliquant sur le bouton correspondant, une boite de dialogue vous demander une confirmation pour la validation de cet évènement.

Editer un évènement

Pour éditer un évènement il est nécéssaire de se rendre sur la page de consultation du calendrier des évènements et de cliquant sur une des cases du tableau, une boite de dialogue s'ouvre alors:

Il suffit alors de modifier les champs présents à votre guise.
Pour valider la modification de l'évènement, cliquez sur le bouton "Modifier l'évènement".
