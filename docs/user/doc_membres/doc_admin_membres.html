Administration des membres

  • Valider ou supprimer une/des inscription(s)

Valider une inscription:

La présence d'une notification en rouge vous informe qu'il y a des inscriptions à valider:

Pour valider une inscription, cela se passe en deux étapes:

Tout d'abord, rendez-vous sur la page d'administration des membres en cliquant sur l'icône adéquate dans le panel d'administration du site. Ensuite sélectionnez le traitement "Valider les inscription" dans la liste déroulante des traitements comme ceci :

Maintenant selectionnez sur les cases à cocher en face de chaque inscription à valider ou supprimer. Puis cliquez sur le bouton valider ou supprimer selon le choix de l'action à effecuter, le cas échéant une demande de confirmation apparaitra à l'écran pour la suppression. Enfin un message de confirmation vous informera du bon déroulement de l'opération.

Editer un profil de membre:

Pour éditer le profil d'un joueur, cela se passe en trois étapes:

Tout d'abord, rendez-vous sur la page d'administration des membres en cliquant sur l'icône adéquate dans le panel d'administration du site. Ensuite sélectionnez le traitement "Editer un membre" dans la liste déroulante des traitements comme ceci :

Maintenant selectionnez le profil a éditer parmis la liste de membres validés du site. Puis cliquez sur le bouton éditer pour accéder à la page d'édition du profil. A partir de là vous pouvez modifier toutes les informations du profil du membre. Enfin un message de confirmation vous informera du bon déroulement de l'opération, une fois que vous aurez valider la oules modification(s).
