Tout d'abord rendez-vous sur la page d'inscription du site, en cliquant sur le lien "Créer un compte" en haut au milieu de la barre noire:

Maintenant remplissez tous les champs munis d'une *, car ils sont obligatoires. Certains champs pourront-êtres édités ultérieurement. Néanmoins ils doivent-êtres remplis consciensieusement notamment pour la question secrète qui permettra la récupération de votre mot de passe en cas d'oubli. Pour le code de sécurité, vous devez remplir la champ avec le numéro en vis-à-vis.

Comment changer son mot de passe ?

Pour changer son mot de passe actuel, cela se passe en deux étapes:

Tout d'abord rendez-vous sur la page d'édition du profil, en cliquant sur l'icone "Profil" dans le panel membre: Maintenant remplissez tous les champs de l'encadré "Mot de passe" de la page, puis cliquez sur le bouton valider. Une demande confirmation apparaitra. Une fois celle-ci confirmée, un message de confirmation vous informera du bon déroulement de l'opération.

Comment changer sa photo de profil ?

Pour changer sa photo de profil, cela se passe en deux étapes:

Tout d'abord rendez-vous sur la page d'édition du profil, en cliquant sur l'icone "Profil" dans le panel membre. Maintenant cliquez sur l'image actuelle qui vous est attribuée, une fenêtre s'ouvre vous demandant de selectionner une nouvelle photo. Selectionnez-en une puis cliquez sur envoyer. Attention le temps de chargement peut être long tout dépends de votre connexion internet! Une fois le chargement terminé, un message de confirmation vous informera du bon déroulement de l'opération. Néanmoins il se peut que vous ne puissiez pas la voir desuite à cause de la mémoire temporaire de votre navigateur. Dans ce cas patientez un petit peut et cela viendra.
