Catégories

Administration des modules "club" et "ultimate"

  • Ajouter une section
  • Supprimer une section

Ajouter une section:

Pour ajouter une section à l'un de ces deux modules, il faut tout d'abord, se rendre sur la page d'un des deux module en cliquant dans le menu sur d'une des deux catégories. Ensuite, cliquer sur le bouton + en bas la liste des sections disponibles.

Maintenant remplissez le champ avec le nom que vous désirez. Puis cliquez sur le bouton valider. La page se rechargera et la nouvelle section sera disponible.

Suppression une section:

Pour supprimer une section à l'un de ces deux modules, il faut tout d'abord, se rendre sur la page d'un des deux module en cliquant dans le menu sur d'une des deux catégories. Ensuite, cliquer sur le bouton - en bas la liste des sections disponibles.

Maintenant sélectionnez le nom de la section que vous voulez voir disparaitre. Puis cliquez sur le bouton valider. La page se rechargera et la section sera supprimée.
